# Claude Workbench

> 一个集成了 Claude Code 的可视化工作台，让 AI 助手与内容预览无缝衔接。

## 功能特性

- **终端集成** — 内嵌 Claude Code 终端，无需切换窗口
- **内容预览** — 文档、图片、网页、PPT 等直接在工作台预览
- **Team 状态监控** — 任务进度和异常告警一目了然
- **文件监听** — 指定文件夹新文件自动出现
- **VS Code 风格布局** — 左导航 + 中终端 + 右预览，高效协同

## 技术栈

- **Tauri 2.x** — 轻量、安全的桌面框架
- **React 18 + TypeScript** — 前端框架
- **TailwindCSS** — 样式解决方案
- **xterm.js + node-pty** — 嵌入式终端
